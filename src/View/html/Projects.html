Landon Hotel (Used SpringBoot, Angular, Java, SQL, TypeScript, HTML, and CSS)
In this project, I created an online booking website for the fictitious "Landon Hotel." This website allowed users to book an available room, which would then be stored in an sql database for the given dates. After booked, the website would not allow other users to book the room during the timeframe

Android Vacation App (Used Android Studio, Java, and XML)
This project had me develop an Android app that allowed users to plan vacations. As part of the planning process, users were able to add, edit, and delete vacations through a GUI that I developed. This GUI consisted of 2 screens, 1 to view, edit, and delete vacations from the list of added vacation, and another to add new vacations. This app also allowed users to enable notifications for the start and end dates of their vacations.
